Como a Lumen Transforma Texto em Tato

Nosso sistema é uma ponte eficiente e de baixo custo entre a tecnologia digital e a leitura tátil, utilizando comunicação em tempo real.

  • 1.

    Tradução Web

    O usuário digita o texto no nosso site. O JavaScript traduz o texto instantaneamente para Braille Grau 1 (Unicode).

  • 2.

    Comunicação WebSocket

    A string Braille é enviada por **WebSocket** para o ESP32. Esta conexão persistente garante que o comando chegue à caixa sem atrasos.

  • 3.

    Processamento no ESP32

    O ESP32 recebe a string, quebra o texto caractere por caractere e converte cada célula Braille em um **padrão binário** (0s e 1s).

  • 4.

    Atuação Física

    O ESP32 envia sinais (HIGH/LOW) para os drivers de motor/solenoides conectados, que levantam ou abaixam os pinos da célula Braille, permitindo a leitura tátil da palavra.

Fluxo de Dados

Navegador (Texto)
WebSocket (Braille)
ESP32 (Controle)
Célula Tátil (Saída)
